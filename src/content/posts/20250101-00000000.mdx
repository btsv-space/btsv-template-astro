This is a blog powered by [btsv](https://github.com/btsv/btsv) — a markdown+ editor paired with a static site builder. Write posts in a local-first editor, save them to git, and publish to any static host.

Every post lives as a markdown file in your repository. You own your content. No vendor lock-in, no database, no platform that can disappear overnight.

## Features

- GitHub-flavored Markdown with MDX support
- Callouts, figures, and editor-only comments
- Automatic light and dark mode
- Syntax highlighting for code blocks
- Static output — deploy anywhere
